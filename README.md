  # 💻 GSAP Macbook Pro Landing Page - 3D Experience

Una landing page premium inspirada en Apple, diseñada para ofrecer una experiencia inmersiva mediante el uso de modelos 3D interactivos y animaciones de alto rendimiento.

## ✨ Características Principales

- **Interacción 3D:** Renderizado de un Macbook Pro detallado utilizando @react-three/fiber y @react-three/drei.
- **Animaciones de Scroll:** Uso extensivo de GSAP (ScrollTrigger) para transiciones fluidas y efectos de revelado.
- **Diseño Responsivo:** Adaptabilidad total a dispositivos móviles y escritorio usando Tailwind CSS.
- **Rendimiento Optimizado:** Construido sobre Vite para tiempos de carga ultrarrápidos.
